Unblock Mode
Unblock: Unblock strands aligned to the target region.

Enrich: Unblock strands that not aligned to the target region.

Balance: Balance coverage of target regions. (Not available)

Barcodes
"None":
apply action to all reads with/without barcodes

"All":
apply action to all reads with successfully assigned barcodes

Barcodes (split by comma):
apply action to reads with specific barcodes

"Unclassified":
apply action only to unclassified reads

Target region
chr:start-end
apply action to reads aligned to target region

chr
apply action to reads aligned to target chromosome

"Mapped"
apply action to reads aligned to reference transcriptome

"Unmapped"
apply action to unmapped reads

Jobs batch upload (only support JSON format)
Example:

{
    "time|0-60": {
        "ch|0-128": {
            "bc|all": {
                "target|chrM": "Enrich",
                "target|chr21": "Unblock",
            }
        },
        "ch|128-256": {
            "bc|all": {
                target|/tmp/config/targets.bed": "Unblock"
            }
        }
    }
}
